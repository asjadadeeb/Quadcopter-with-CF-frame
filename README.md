# Shaastra Aerial Robotics Challenge — Custom Pixhawk Quadcopter

## Project Overview

Welcome to the project repository for my custom Pixhawk 2.4.8 quadcopter, engineered specifically to compete in the **Shaastra Aerial Robotics Challenge**. This repository serves as complete documentation of the drone's development, featuring a test flight video, custom fabrications.

To meet the rigorous demands of the competition, the drone was built entirely from scratch. The core of the design is a custom-engineered, lightweight H-frame constructed from carbon fiber rods integrated with repurposed F330 PCB plates, further protected by custom-designed propeller guards.

---

## Hardware Specifications

* **Flight Controller:** Pixhawk 2.4.8
* **GPS Navigation:** M9N GPS Module
* **Power Management:** 3DR Power Module
* **Motors:** 2304 2300KV BLDC 
* **Propellers:** 5-inch with Custom-Designed Prop Guards 
* **Battery:** Custom-built 4S 5000mAh Li-ion pack (built using Samsung 21700 cells)
* **Frame:** Custom hybrid carbon fiber rod and F330 PCB plate design (H-Configuration)

---

## Build Gallery & Design Details

### 🛠️ Frame Construction
The custom H-configuration frame was designed to keep the overall width to a minimum while maintaining high structural rigidity. The arms are constructed using **10mm pultruded carbon fiber (CF) rods** mounted to repurposed F330 PCB plates.
